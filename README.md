# filefly wallet 恒星钱包


系统部署说明
----------------
> 1. 安装nodejs
> 2. clone 代码
> 3. 在项目目录，输入npm install(可以使用npm install -g cnpm ，然后再输入cnpm install)
> 4. npm run dev可以预览项目
> 5. npm run build 打包
> 6. npm run android，预览android工程



主要业务说明
----------------
> 1. 如果是已经有钱包地址，但没有输入密码，那么账户是只读状态，不能发送，但对接收
> 2. 默认只查询出200条数据
